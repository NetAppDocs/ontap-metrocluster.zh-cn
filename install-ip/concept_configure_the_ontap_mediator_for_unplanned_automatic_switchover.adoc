= 为计划外自动切换配置 ONTAP 调解器服务
:icons: font
:imagesdir: ../media/


[role="lead"]
从 ONTAP 9.7 开始， ONTAP 调解器服务可以通过为状态信息提供物理上独立的存储库来协助 MetroCluster IP 配置执行自动计划外切换。

--
[NOTE]
====
* ONTAP 调解器服务和 MetroCluster Tiebreaker 软件不应同时使用相同的 MetroCluster 配置。
* ONTAP 调解器服务最多可同时支持五个 MetroCluster 配置。


====
