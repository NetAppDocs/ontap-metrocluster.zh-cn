= 维护 MetroCluster 组件
:icons: font
:imagesdir: ../media/


[role="lead"]
本指南介绍如何对 MetroCluster 硬件组件执行维护。

此内容介绍了 MetroCluster 系统的维修过程，包括驱动器架， IP 交换机， FC 交换机， FC-SAS 网桥和 SAS 缆线的固件升级和硬件更换过程。

您应使用本指南维护 MetroCluster 配置中的 MetroCluster 专用硬件组件。
