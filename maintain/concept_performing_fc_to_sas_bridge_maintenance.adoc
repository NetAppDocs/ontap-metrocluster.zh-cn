= 执行 FC-SAS 网桥维护
:icons: font
:imagesdir: ../media/


[role="lead"]
如有必要，您可以在 MetroCluster 配置中无中断更换 FC-SAS 网桥或升级其固件。


NOTE: 运行 ONTAP 9.8 及更高版本的配置不支持 FibreBridge 6500N 网桥。
