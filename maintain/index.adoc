= 维护 MetroCluster 组件
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用以下过程维护 MetroCluster 配置中的 MetroCluster 专用硬件组件。

MetroCluster 系统的服务过程包括驱动器架， IP 交换机， FC 交换机， FC-SAS 网桥和 SAS 缆线的固件升级和硬件更换过程。
