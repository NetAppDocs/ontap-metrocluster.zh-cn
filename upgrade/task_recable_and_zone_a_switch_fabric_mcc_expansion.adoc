= 为新节点的交换机网络结构重新布线和分区
:icons: font
:imagesdir: ../media/


[role="lead"]
在向 MetroCluster 配置添加节点时，您必须更改布线，然后运行 RCF 文件以重新定义网络结构上的分区。

必须对每个交换机网络结构执行此任务。它一次只能由一个网络结构完成。
