= 为新控制器模块布线和 #8217 的集群对等连接
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须使用缆线将新控制器模块连接到集群对等网络，以使其与配对站点上的集群建立连接。

每个控制器模块上至少应使用两个端口建立集群对等关系。

端口和网络连接的建议最小带宽为 1 GbE 。

.步骤
. 确定至少两个端口并为其布线以建立集群对等关系，然后验证它们是否与配对集群建立了网络连接。
