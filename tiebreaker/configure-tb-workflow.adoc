= MetroCluster Tiebreaker 配置工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
安装 Tiebreaker 软件后，您可以添加 MetroCluster 配置、配置 SNMP 设置并开始监控您的 MetroCluster 环境。

.关于此工作流
您可以在安装后使用此工作流程配置 Tiebreaker 软件。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一个"]link:concept_configuring_the_tiebreaker_software.html["配置 Tiebreaker 软件"]
[role="quick-margin-para"]
向 Tiebreaker 软件添加 MetroCluster 配置、导入证书和配置监控设置。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["两个"]link:task_configure_snmp_setts_for_tiebreaker_software.html["配置 MetroCluster Tiebreaker 软件的 SNMP 设置"]
[role="quick-margin-para"]
配置 SNMP 设置，以便 Tiebreaker 软件可以发送 MetroCluster 事件的 SNMP 通知。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三个"]link:concept_monitoring_the_mcc_configuration.html["监控 MetroCluster 配置"]
[role="quick-margin-para"]
使用 Tiebreaker 软件监控 MetroCluster 配置状态、查看 SNMP 事件和管理 AutoSupport 设置。
