= 启动灾难站点上的设备（ MetroCluster IP 配置）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须打开灾难站点上的磁盘架和 MetroCluster IP 交换机组件的电源。灾难站点上的控制器模块将保留在 LOADER 提示符处。

此操作步骤中的示例假设如下：

* 站点 A 是灾难站点。
* 站点 B 是正常运行的站点。


.步骤
. 打开灾难站点上的磁盘架，并确保所有磁盘都在运行。
. 如果尚未打开 MetroCluster IP 交换机，请将其打开。
