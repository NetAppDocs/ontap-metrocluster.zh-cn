= ONTAP Mediator 对MetroCluster的支持有哪些新特性
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
了解MetroCluster为ONTAP Mediator 支持带来的新功能和增强功能。

有关ONTAP Mediator 每个版本的功能和增强特性的详细信息，请参阅：link:https://docs.netapp.com/us-en/ontap/mediator/whats-new-concept.html["ONTAP Mediator 的新增功能"^] 。

[cols="4,2"]
|===
| ONTAP调解器功能 | 可从一开始使用 


 a| 
在MetroCluster IP 配置中， ONTAP Mediator 1.11 或更高版本支持 IPv6。

link:../install-ip/task_configuring_the_ontap_mediator_service_from_a_metrocluster_ip_configuration.html["为MetroCluster IP 配置设置ONTAP调解器"]
 a| 
ONTAP 9.18.1



 a| 
ONTAP Mediator 1.11 增加了对使用单个ONTAP Mediator 实例管理多达 10 个MetroCluster IP 配置的支持。

link:../install-ip/concept_mediator_requirements.html["准备在MetroCluster IP 配置中安装ONTAP Mediator"]
 a| 
ONTAP 9.18.1



 a| 
在环境关闭时、支持调解器辅助的自动计划外切换(MAUSO)。

如果一个站点因环境关闭而正常关闭、则会触发MAUSO。

https://docs.netapp.com/us-en/ontap-metrocluster/install-ip/concept-ontap-mediator-supports-automatic-unplanned-switchover.html["ONTAP 调解器如何支持自动计划外切换"]

ONTAP 9.13.1



 a| 
MetroCluster IP 配置中对 ONTAP 调解器的初步支持
 a| 
ONTAP 9.7
