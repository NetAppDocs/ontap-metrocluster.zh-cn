= 为 MetroCluster 站点之间的 ISL 布线
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须通过光纤交换机间链路（ ISL ）连接每个站点的 FC 交换机，以形成连接 MetroCluster 组件的交换机网络结构。

必须同时对两个交换机网络结构执行此操作。

.步骤
. 使用表中与您的配置和交换机型号对应的布线方式，将每个站点的 FC 交换机连接到所有 ISL 。
