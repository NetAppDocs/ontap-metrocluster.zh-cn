= 规划和安装使用阵列 LUN 的 MetroCluster 配置
:icons: font
:imagesdir: ../media/


[role="lead"]
如果要在 MetroCluster 配置中使用阵列 LUN ，则必须规划安装并按照此类配置的特定过程进行操作。您可以将 MetroCluster 配置设置为混合使用阵列 LUN 和原生磁盘架，也可以仅使用阵列 LUN 。
