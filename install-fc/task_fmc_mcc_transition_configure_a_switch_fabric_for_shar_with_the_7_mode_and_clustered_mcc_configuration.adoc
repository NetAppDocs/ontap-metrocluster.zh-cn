= 在 7- 模式和集群模式 MetroCluster 配置之间配置交换机网络结构共享
:icons: font
:imagesdir: ../media/


[role="lead"]
要在现有的 7- 模式网络结构 MetroCluster 和新的 MetroCluster 配置之间共享交换机网络结构，您必须设置与非共享配置不同的特定分区和其他设置。

必须对两个交换机网络结构执行此任务，一次一个。
