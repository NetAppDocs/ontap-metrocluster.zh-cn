= 在使用阵列 LUN 的配置中安装 MetroCluster 组件并为其布线
:icons: font
:imagesdir: ../media/


[role="lead"]
要设置使用阵列 LUN 的 MetroCluster 配置，您必须使用缆线将存储控制器连接到 FC 交换机，并为 ISL 布线以链接站点。此外，您还必须使用缆线将存储阵列连接到 FC 交换机。
