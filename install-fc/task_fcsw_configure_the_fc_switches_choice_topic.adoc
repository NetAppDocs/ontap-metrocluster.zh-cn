= 配置 FC 交换机
:icons: font
:imagesdir: ../media/


[role="lead"]
对于出厂时未预先配置的光纤连接 MetroCluster 系统，您必须配置 DR 组中的每个 FC 交换机。这可以手动完成，也可以根据交换机使用配置文件来完成。

对于新系统， FC 交换机网络结构通常为两个 ISL 进行配置，并且不需要进行额外配置，除非您要更改预先配置的 IP 地址。
