= 使用阵列 LUN 的 MetroCluster 配置中的硬件组件机架安装
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须确保使用阵列 LUN 设置 MetroCluster 配置所需的硬件组件已正确装入机架。

您必须在两个 MetroCluster 站点上执行此任务。

.步骤
. 规划 MetroCluster 组件的定位。
+
机架空间取决于存储控制器的平台型号，交换机类型以及配置中的磁盘架堆栈数量。

. 正确接地。
. 在机架或机柜中安装存储控制器。
+

NOTE: 阵列 LUN 不支持 AFF 系统。
