= 使用阵列 LUN 的 MetroCluster 配置中的交换机分区
:icons: font
:imagesdir: ../media/


[role="lead"]
通过配置交换机分区，您可以定义 MetroCluster 配置中的特定 ONTAP 系统可以查看哪些阵列 LUN 。

交换机分区用于定义已连接节点之间的路径。通过配置分区，您可以定义特定 ONTAP 系统可以查看哪些阵列 LUN 。
