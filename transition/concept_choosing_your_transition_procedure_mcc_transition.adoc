= 选择过渡操作步骤
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须根据现有 MetroCluster FC 配置选择过渡操作步骤。

过渡操作步骤将后端 FC 交换机网络结构或 FC-VI 连接替换为 IP 交换机网络。确切的操作步骤取决于您的初始配置。

在过渡操作步骤结束时，原始平台和 FC 交换机（如果存在）将停用。
